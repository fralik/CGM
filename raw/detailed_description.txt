========================== en ==========================

After you install the extension, new icon will appear on the toolbar. Simply click this icon when you are on any Google page and have fun rearranging the links! If the Google page was open before you have installed the extension, you should reload it.

Make sure you allow interaction with secure pages in extension's privacy settings! Otherwise the extension may not work.

Use the options page to hide the extension icon from the toolbar. 

Please report any issues or feature requests on extension user-group (http://groups.google.com/group/cgm-users/). Source code and additional README file are available at https://github.com/fralik/CGM/

========================== ru ==========================

После установки приложения на панели инструментов появится новый значок. Щелкните мышкой по этому значку, когда вы находитесь на какой–нибудь странице Гугла, и вы сможете начать изменять структуру меню. Если страница Гугла была открыта еще до установки расширения, то перезагрузите её.

Убедитесь, что вы разрешили доступ к защищенным страницам в настройках конфиденциальности расширения. В противном случае расширение может не работать.

Вы можете спрятать значок расширения с панели инструментов, воспользовавшись страницей настроек. 

Предложения по улучшению расширения или информацию о багах, пишите, пожалуйста, в группу по адресу http://groups.google.com/group/cgm-users/. Исходный код и расширенный файл с описанием находятся на странице разработки расширения (https://github.com/fralik/CGM/).

========================== de ==========================
Nachdem Sie die Erweiterung installiert haben, wird ein neues Symbol auf der Symbolleiste angezeigt. Klicken Sie einfach auf dieses Symbol, wenn Sie auf der Google™ Seite sind. Viel Spaß beim Erstellen der neuen Ordnung der Links! Wenn die Google™ Seite schon offen war, bevor Sie die Erweiterung installiert haben, sollten Sie die Seite neu laden.

Stellen Sie sicher, dass die Interaktion mit geschtüzten Seiten in der Privatsphäre-Einstellungen der Erweiterung erlaubt ist, sonst funktionieret die Erweiterung nicht.

Um das Symbol der Erweiterung von der Symbolleiste auszublenden, verwenden Sie die demendsprechende Option. 

Haben Sie Kommentare oder Wünsche? Melden Sie sich unter http://groups.google.com/group/cgm-users/. Die Quellcode und weitere Informationen können Sie unter https://github.com/fralik/CGM/ finden.

========================== es ==========================
El nuevo icono aparecerá en la barra de herramientas después de instalar la extensión. Simplemente hace clic en este icono en la página de Gmail™ y siga reorganizar los enlaces. Si la página de GMail™ fue abierta antes de instalar la extensión, hay que volver a cargarla.

Asegúrese que la interacción con las páginas seguras está permitida en la configuración de privacidad de extensión. La extensión puede no funcionar sin el permiso.

Se puede ocultar el icono de la extensión en la página de las configuraciónes.

Por favor informen de los problemas o sugieran nuevas funciones en el grupo de la extensión (http://groups.google.com/group/cgm-users/).
Se puede ver el código fuente y un README adicional en https://github.com/fralik/CGM/
